Carga de perfil

Una vez que el usuario ha almacenado alguna configuración, se genera un perfil. Se puede almacenar la configuración de accesibilidad deseada, la configuración de la firma y también la configuración general de la herramienta. A partir de ese momento, cada vez que se arranque la herramienta se mostrará la ventana de selección del perfil. Tras seleccionar el perfil, la herramienta se cargará con la configuración de accesibilidad y general almacenada. Existe también el perfil "Predeterminado", para el cual no se activa ninguna opción de accesibilidad ni ninguna configuración general.

La pantalla de carga de perfil es la mostrada a continuación:

Los elementos disponibles en esta pantalla son:

  • Perfil a cargar: Seleccionar el perfil que se quiere cargar. Por defecto está seleccionado el último perfil usado.
    • Tecla de acceso rápido: p

Al pulsar el botón Aceptar se cargará la herramienta con la configuración del perfil seleccionado.

  • Tecla de acceso rápido: a

Al pulsar el icono de ayuda se muestra la ventana de ayuda para la pantalla.

  • Tecla de acceso rápido para el botón de Ayuda: h
