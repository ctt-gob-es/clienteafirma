Contrafirma

La contrafirma o firma en cascada consiste en la firma de una firma anterior de un documento, de tal forma que se muestra el acuerdo o la comprobación de esta firma. Todas las firmas realizadas se almacenan en el mismo documento de firmas. En este tipo de firmas sí importa el orden en el que se realizan las firmas.

Para realizar esta operación es necesario que se realice una primera firma simple del documento para, a continuación, generar las contrafirmas a partir de ella. Para contrafirmar una firma no es necesario disponer del documento original de datos, basta con el documento con las firmas que se desean contrafirmar.

Algunos formatos de firma como PDF, ODF y OOXML no admiten contrafirmas.
