Firma XML

Los formatos de firma XML son aquellos que organizan la información contenida en la firma en estructuras XML, legibles directamente como texto plano.

Los formatos de firma XML más extendidos son:

  • XMLdSig (Firma simple)
  • XAdES (Firma avanzada)

Adicionalmente, existen formatos de documento para los que se ha estandarizado un mecanismo de firma basado en firmas XML. Ejemplos de estos tipos de firma son:

  • ODF (Firma simple)
  • OOXML (Firma simple)

La Interfaz de Escritorio del Cliente @firma sólo permite la generaración de firmas en estos formatos cuando se activa la vista avanzada de la herramienta.
