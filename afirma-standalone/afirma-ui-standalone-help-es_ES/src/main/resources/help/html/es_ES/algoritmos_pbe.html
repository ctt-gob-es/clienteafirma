Algoritmos PBE

Para facilitar el uso del cifrado simétrico se crearon los algoritmos denominados PBE. Estos algoritmos permiten el cifrado simétrico de datos por medio a una contraseña formada por caracteres ASCII. De esta forma, el usuario puede cifrar y descifrar datos sin necesidad de almacenar una clave de cifrado binaria en disco.

Los algoritmos soportados por el Cliente @firma para el cifrado simétrico basado en contraseña son:

  • PBEWithSHA1AndDESede (Recomendado)
  • PBEWithMD5AndDES
  • PBEWithSHA1AndRC2_40
