Introducción

La interfaz de escritorio de @firma nace por el deseo del Ministerio de la Presidencia de proporcionar a los usuario una herramienta que les permita de forma sencilla y gratuita llevar a cabo la firma electrónica de documentos por motivos personales o profesionales, sin que exista relación directa con ningún trámite electrónico. La interfaz de escritorio de firma forma parte del proyecto Cliente @firma y da acceso a parte de las funcionalidades del propio Cliente.

La interfaz de escritorio incorpora las siguientes funcionalidades:

  • Firma electrónica
  • Multifirma electrónica
  • Cifrado de datos
  • Descifrado de datos
  • Generación de sobres digitales
  • Apertura de sobres digitales

Al contrario que el Cliente @firma, la interfaz de escritorio puede ejecutarse sin necesidad de un navegador ni conexión a Internet.
