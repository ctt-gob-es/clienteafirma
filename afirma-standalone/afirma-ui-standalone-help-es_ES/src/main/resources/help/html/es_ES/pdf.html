PDF

El formato PDF / ISO 32000-1 define el soporte de firmas mediante el algoritmo RSA y huellas digitales según algoritmos SHA-1 y SHA-2. Este tipo de firma consiste, básicamente, en la inclusión de una estructura de firma CAdES en un espacio reservado dentro del documento PDF que se firma.

Las firmas PDF que generan el Cliente @firma y su Interfaz de Escritorio siguen las especificaciones establecidas por el estándar PAdES. Así pues, seleccionese en la interfaz el formato PAdES para generar firmas de documentos PDF.

Tal como se generan en el Cliente y la Interfaz, estas firmas no tienen representación gráfica dentro del documento.
