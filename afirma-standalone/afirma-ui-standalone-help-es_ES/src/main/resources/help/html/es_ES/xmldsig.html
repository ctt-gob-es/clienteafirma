XMLdSig

Formato de firma XML desarrollado por el W3C. Este formato puede ser aplicado a cualquier tipo de contenido digital y permite la generación de la firma de uno o más objetos de datos.

El formato define distintos tipos de organización para el documento de firma:

  • Detached: Firma de un documento externo al XML de firma. De este tipo pueden distinguirse los siguientes casos:
    • Internally Detached : Cuando el objeto firmado está dentro de un mismo contexto XML que los datos firmados. Esto es, un XML que engloba los elementos de firma y datos manteniéndolos como nodos hermanos.
    • Externally Detached: Cuando el elemento XML de firma y el documento firmado no comparten contexto.

  • Enveloping: Firma de un documento interno al XML de firma.
  • Enveloped: Firma del documento XML que contiene la firma. Sólo válido para firmar documentos XML.

La Interfaz de Escritorio del Cliente @firma no permite la generación de firmas XMLdSig. En su lugar, permite la generación de su variante de firma avanzada, XAdES.
