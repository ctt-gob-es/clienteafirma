XAdES

Estándar de firma avanzada XML que agrega restricciones y extiende al formato XMLdSig.

El formato XAdES Detached con el modo de firma implícito es el recomendado por las Administraciones Públicas españolas para la generación de firmas.

El Cliente @firma permite realizar firmas XAdES-BES (firmas simples) y XAdES-EPES (firmas con política de firma externa) acordes a las versiones 1.3.2 (por defecto) y 1.4.1 del estándar.

La versión actual de la Interfaz de Escritorio sólo soporta la generación de firmas XAdES-BES conforme a la versión 1.3.2 del estándar.
