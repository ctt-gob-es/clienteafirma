Política de firma

Conjunto de reglas que definen los requisitos técnicos y procedimentales para la creación y validación de firmas electrónicas, con el objetivo de satisfacer una necesidad de negocio particular y, bajo la cual, la firma puede ser determinada como válida.

Los formatos de firma avanzada pueden implementar dos mecanismos para el establecimiento de una política, mantener las políticas incrustada en la firma o mantener únicamente una referencia a la política. Esta última opción es la adoptada por todos los formatos de firma soportados por el Cliente @firma: firmas EPES (Explicit Policy Electronic Signature).

La interfaz de escritorio del Cliente @firma soporta la configuración de políticas a través del diálogo de configuración. Acceda al apartado Opciones de configuración para más información.
