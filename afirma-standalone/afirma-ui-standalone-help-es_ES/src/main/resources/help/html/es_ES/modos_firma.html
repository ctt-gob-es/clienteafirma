Modos de firma

El modo de firma hace referencia a la disposición de los datos firmados con respecto a la firma generada. él concepto "modo de firma" puede no estar contemplado por todos los formatos de firma pero, por simplificar, se distinguen 2 modos de firma que conceptualmente se pueden aplicar de manera informal a cualquier formato:

  • Modo implícito: Se dice que una firma es implícita cuando el documento de firma generado contiene el documento que se ha firmado. Las distintas distribuciones contempladas por esta definición son:
    • El documento firmado se incorpora dentro de la estructura de firma generada.
    • La estructura de firma generada pasa a formar parte del documento firmado.
    • Se genera una nueva estructura que alberga tanto al objeto de firma como al documento firmado.
  • Modo explícito: Se dice que una firma es explícita cuando genera un documento de firma que no incorpora el documento que se firmó.

Existen formatos de firma que admiten los modos de firma implícito y explícito, mientras que otros no contemplan el concepto de "modo de firma". Estos últimos generarían siempre firmas acorde a uno de estos modos o serían lo suficientemente flexibles como para generar firmas de ambos tipos.
