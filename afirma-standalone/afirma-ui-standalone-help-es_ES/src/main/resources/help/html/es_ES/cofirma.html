Cofirma

La cofirma o firma paralela consiste en la firma de un mismo documento por parte de varios firmantes, de tal forma que todos muestran estar de acuerdo con su contenido. Todas las firmas realizadas se almacenan en el mismo documento de firmas. En este tipo de firmas no importa el orden en el que se realizan las firmas.

Para realizar esta operación es necesario que se realice una primera firma simple del documento para, a continuación, generar las cofirmas y almacenar el resultado en el documento de firma generado. La cofirma de datos requiere tanto los datos que se quieren firmar como el fichero con las firmas previas.

En el caso de algunos formatos de firmas como PDF, ODF y OOXML se incrustan en el documento de firmado las distintas firmas independientes generadas, de tal forma que la operación de cofirma equivale a la de firma simple. En estos casos suele hablarse generalmente de multifirma.
