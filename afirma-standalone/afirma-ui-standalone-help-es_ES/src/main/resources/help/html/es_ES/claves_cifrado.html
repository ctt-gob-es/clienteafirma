Claves de cifrado

Las claves de cifrado son conjuntos de bits con un formato específico que las hacen compatibles para su uso con un algoritmo concreto de cifrado. Esto hace que no sea posible utilizar como clave para un algoritmo concreto una contraseña cualquiera, salvo que el propio algoritmo esté preparado para ello.

En cifrado asimétrico puede hablarse de claves de cifrado públicas y privadas. Aunque no hay ninguna diferencia a nivel técnico, las claves de cifrado públicas son las que se pueden distribuir, mientras que las privadas son las que las que se deben guardarse y no compartir con nadie.

En los algoritmos de cifrado simétrico se pueden cifrar datos con una clave y descifrarlos con esa misma clave.

En los algoritmos de cifrado asimétrico se pueden cifrar datos con la clave pública y descifrarlos con la privada (base del sobre electrónico) y viceversa (base de la validación de firma).
