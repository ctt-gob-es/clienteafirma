Realizar Firma Electrónica (Firm@Movil)

Una firma electrónica es un conjunto de datos asociados a un mensaje que permite asegurar la identidad del firmante y la integridad del mensaje.

Para realizar la firma del texto, descargue el Cliente móvil @firma y pulse el boton "Firmar".





Nota: La firma electrónica se realiza mediante una aplicación firmada por la D.G. para Impulso de la Administración Electrónica por su seguridad. Pulsa aquí para ver los requisitos de funcionamiento
