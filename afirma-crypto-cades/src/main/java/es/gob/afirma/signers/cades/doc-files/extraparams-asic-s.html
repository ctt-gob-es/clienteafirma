Valores aceptados como parámetros adicionales para las firmas CAdES:

asicsFilename
En las firmas simples CAdES ASiC-S, determina el nombre del fichero de datos dentro del contenedor.
Si no se establece este parámetro se usa el nombre dataobject con la extensión correspondiente al tipo de datos, o .bin si esta primera no puede determinarse de forma automática.
